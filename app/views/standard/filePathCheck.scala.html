How to fix long file path issues
  • Shorten folder or file names in the path
  • Move the folder closer to the root of your drive (e.g. C:\transfers\)
  • Ask your IT team to enable long path support in Windows (registry key: HKLM\SYSTEM\CurrentControlSet\Control\FileSystem\LongPathsEnabled)
