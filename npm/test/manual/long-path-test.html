Check your transfer for long file path issues

Windows browsers may not be able to read files or folders when the full file path exceeds 260 characters. This can cause files to be silently omitted from your transfer.

Use this page to check whether your folder is affected before you start your transfer.

Select the same top-level folder you intend to transfer. This check does not upload any files — it only reads the file paths in your browser.
